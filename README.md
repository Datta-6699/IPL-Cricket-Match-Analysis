# IPL Cricket Match Analysis 🏏

Welcome to the **IPL Cricket Match Analysis** project! This repository contains a comprehensive Exploratory Data Analysis (EDA) of Indian Premier League (IPL) matches. The project aims to uncover key insights, team performances, venue statistics, and player records using modern Data Science tools.

## 🌟 Project Overview
The Indian Premier League (IPL) is a massive phenomenon in the world of cricket. This project leverages historical datasets to perform statistical analysis and answer critical questions such as:
- Which teams are the most successful?
- What impact does winning the toss have on the final match outcome?
- How do venues influence scoring?
- Who are the highest run-scorers and leading wicket-takers?

By cleaning, preprocessing, and visualizing the data, this notebook serves as both an educational walkthrough for data enthusiasts and a professional demonstration of Python-based data manipulation.

## ✨ Features
- **Data Preprocessing**: Handling missing values, cleaning text data, and formatting columns.
- **Exploratory Data Analysis (EDA)**: Grouping, filtering, and statistical analysis using Pandas.
- **Data Visualizations**: Clean, insightful, and appealing charts created with Matplotlib and Seaborn.
- **Actionable Insights**: Clear takeaways derived from data trends (e.g., season-by-season changes, toss advantages).

## 🛠️ Technologies Used
- **Language**: Python 3
- **Data Manipulation**: Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn
- **Environment**: Jupyter Notebook

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Datta-6699/IPL-Cricket-Match-Analysis.git
   cd IPL-Cricket-Match-Analysis
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
   ```

3. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```
   *Then, open `IPL_Match_Analysis.ipynb` in your browser.*
